# Subscription_manager

This application is used to manage subscription contracts. It allows users to store and manage information about subscriptions in a database.

The goal of this application is to provide a simple way to manage subscription contracts and send notifications for renewals.

# Requirements

Before running the application, make sure you have:

A configured database

A .env file containing the database connection properties

Environment Configuration

Create a .env file in the root directory of the project and add the required database configuration.

DB_HOST
DB_PORT
DB_USER
DB_PASSWORD
DB_NAME
JWT_SECRET
